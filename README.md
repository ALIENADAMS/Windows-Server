# Windows-Server

# Prosta infrastruktura sieciowa oparta o Windows Server 2022

## Opis projektu
Projekt przedstawia konfigurację podstawowej infrastruktury sieciowej w środowisku wirtualnym, zbliżonej do małej firmy.
Celem było zbudowanie działającego środowiska domenowego wraz z automatycznym przydzielaniem adresów IP oraz redundancją usług.

## Zastosowane technologie
- Windows Server 2022
- Active Directory Domain Services
- DNS
- DHCP
- Ubuntu (klient)

## Funkcjonalności
- Utworzenie domeny Active Directory
- Konfiguracja serwera DNS
- Konfiguracja serwera DHCP

## Architektura
- Windows Server 2022
- Ubuntu

Projekt zawiera zrzuty ekranu jako dowód na działanie wdrożonych funkcji.
